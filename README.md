# Password Strength Checker

This project evaluates how strong a password is based on security rules.

## Features
- Checks password length
- Checks for numbers, letters, and symbols
- Classifies password as Weak / Medium / Strong

## Purpose
To understand password security and improve awareness of secure password creation.

## Technologies Used
- Python

## What I Learned
- Password security rules
- Basic string analysis
- Importance of strong authentication
